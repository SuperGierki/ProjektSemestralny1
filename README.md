# Instrukcje

1. Klonowanie repozytorium, w pożądanej lokalizacji:

`git clone https://github.com/SuperGierki/ProjektSemestralny1.git`

2. Utworzenie wirtualnego środowiska Python:

`python -m venv .venv`

3. Zainstalowanie niezbędnych modułów:

`pip install -r requirements.txt`

4. Utworzenie nowego brancha:

`git branch [nazwa]`

5. Zmiana na nowego brancha:

`git checkout [nazwa]`
